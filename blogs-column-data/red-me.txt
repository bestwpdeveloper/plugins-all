== Description ==
A simple and nice plugin to see some blogs column. And also it'll be helpfull for knowing every blog ID's and you'll that how mutch word have in this blogs and also you can filter your blogs with thumbnails base. This plugin will be helpfull for a admin or a developers. Case he can see the blogs id's easyly.

Unblocking a user is easy. Assign them to their previous role and you're done!

== Installation ==
Installation is fairly straight forward. Install it from the WordPress plugin repository.
